# NoteSnap

NoteSnap is a web application that provides a convenient window for users to create and manage notes. It allows users to add titles and content to their notes, enabling them to organize their thoughts, tasks, or any important information in a structured manner.

## Features

- **User-friendly Interface:** NoteSnap offers an intuitive user interface, making it easy for users to add and delete notes.
- **Title and Content Management:** Users can enter a title and corresponding content for each note, facilitating easy categorization and retrieval of information.
- **Seamless Note Creation:** NoteSnap enables users to create new notes quickly, ensuring a smooth and efficient note-taking experience.

## Technologies Used

- **React:** A JavaScript library for building user interfaces.
- **CSS:** Used for styling the application.
- **JavaScript:** Used for implementing the application's functionality.
- **HTML:** The markup language used for structuring the application
